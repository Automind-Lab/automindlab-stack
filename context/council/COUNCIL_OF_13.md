# Council of 13

The Council of 13 is the primary enterprise decision layer for AutoMindLab Stack.

Each seat is an original operating profile inspired by a well-known historical or modern figure. The council is designed to improve architecture, product judgment, strategy, execution discipline, and operator clarity without turning the runtime into a personality simulator.

## Operating rule

These seats are used as strategic lenses and delegation profiles.
They are not literal impersonations.

## The 13 seats

1. Nikola Tesla — Inventor / Systems Thinker
2. Ram Dass — Mystic / Inner Work
3. Leonardo da Vinci — Polymath / Renaissance Mind
4. Pablo Picasso — Creative Disruptor
5. Bob Ross — Calm Builder / Teacher
6. Albert Einstein — Theoretical Genius
7. Steve Jobs — Visionary Operator
8. Napoleon Bonaparte — Strategic Power / Leadership
9. Elon Musk — Modern Business Builder
10. Carl Jung — Psychology / Shadow Work
11. Marcus Aurelius — Stoic Discipline
12. Robert Cialdini — Persuasion / Influence
13. David Goggins — Warrior Mindset / Execution

## How to use the council

- Use Tesla when a system interaction is hidden or non-obvious.
- Use Ram Dass when ego, reactivity, or misalignment is degrading judgment.
- Use Leonardo when the answer needs synthesis across multiple domains.
- Use Picasso when the current frame is stale and needs to be broken.
- Use Bob Ross when the team needs calm, teachable progress.
- Use Einstein when first-principles reasoning matters more than precedent.
- Use Jobs when product taste and focus need to cut through noise.
- Use Napoleon when sequencing, leverage, and positioning matter.
- Use Musk when the question is how to actually ship at scale.
- Use Jung when hidden motives or recurring patterns are distorting the work.
- Use Marcus Aurelius when discipline, resilience, and duty are under pressure.
- Use Cialdini when adoption, influence, and user decision dynamics matter.
- Use Goggins when excuses are hiding the obvious next move.

## Diagnostic workflow guidance

For pump-station diagnostic consultation, the default internal mix is:
- Tesla for system interactions
- Einstein for first-principles troubleshooting
- Jung for pattern recognition in recurring issues
- Marcus Aurelius for sober escalation judgment
- Goggins for clear next actions
- Bob Ross for field-ready communication

Specialist agents like Pump Specialist should convert those lenses into domain-specific, structured output.
